# Silent Feed（静阅） — 发布说明（v0.1.0）

## 简短描述（适用于商店短描述）
Silent Feed（静阅）——基于本地隐私的智能阅读推荐，帮你安静地发现高质量内容。

## 长描述（适用于 Chrome Web Store 的详细说明）
欢迎使用 Silent Feed（静阅）。

Silent Feed 是一款专注于个人阅读体验的浏览器扩展，通过本地化的行为分析和轻量 AI 适配器，帮助你发现与你兴趣高度匹配的文章与订阅源，同时尽量减少打扰。

主要功能：
- 智能推荐：本地构建用户兴趣画像，优先推荐近期高质量内容；
- 隐私优先：所有敏感数据默认保存在本地，只有在用户明确授权时才上传或同步；
- RSS 支持：自动发现页面中的 RSS/Atom 订阅源并帮助订阅管理；
- 静默通知：克制的提醒机制，避免信息流轰炸；
- 冷启动优化：在首 1000 页面内逐步建立用户画像，提升推荐相关性。

安装与使用：
1. 从 Chrome Web Store 安装扩展或将 `.zip`/`.crx` 手动加载到扩展管理页；
2. 打开扩展图标，完成首次引导（可选择是否允许浏览历史收集）；
3. 访问你常读的网站以加速冷启动；
4. 在设置中管理订阅、隐私与 AI 提供者（如果需要接入第三方服务）。

v0.1.0 亮点（初始发布）：
- 基础推荐引擎与本地画像能力；
- RSS 自动发现与订阅支持；
- 初步的 AI Provider 测试接入（需用户配置 API）；
- 弱化通知策略与界面改进（标准与手绘两种风格）。

已知问题与限制：
- 第三方 AI Provider（若启用）需要用户自行配置 API Key；
- 某些网站的 RSS 自动发现在个别页面可能不准确；
- Chrome Web Store 上线后可能需要根据商店审查意见调整说明文本或权限说明。

## 法律与隐私提示（简要）
Silent Feed 致力于以最小权限运行：默认情况下不上传你的浏览历史或个人数据；任何外发数据都需要在设置中明确授权。如果你对隐私有更高要求，请在设置中关闭数据收集。
