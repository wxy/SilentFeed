## Quick Reference

### Prerequisites
- Node.js 18+
- npm 9+
- Chrome or Edge browser

### Quick Start
```bash
git clone https://github.com/YOUR_USERNAME/SilentFeed.git
cd SilentFeed
npm install
npm run dev
```

### Development Flow
1. Fork the repository
2. Create a feature branch: `git checkout -b feature/xxx`
3. Write code and tests
4. Run `npm run pre-push` to ensure tests pass
5. Commit and create a PR

### Code Standards
- TypeScript strict mode
- Functional components + Hooks
- Test coverage ≥70%
- English comments for code

For detailed information, see [CONTRIBUTING.md](CONTRIBUTING.md).
